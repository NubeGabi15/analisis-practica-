# Analisis de Ventas Retail - Superstore Dataset

## Objetivo
Identificar patrones de ventas, rentabilidad por region y categorias 
con bajo rendimiento para apoyar decisiones comerciales.

## Herramientas usadas
- Python (pandas, matplotlib, seaborn)
- Dataset: Superstore Sales (Kaggle)

## Hallazgos principales
- Technology es la categoria con mas ventas ($836,154)
- La region West lidera en ingresos ($725,457)
- Noviembre es el mes con mas ventas ($352,461) - efecto Black Friday
- Las 3 categorias tienen productos con perdidas por descuentos excesivos

## Graficos
- Ventas por categoria
- Ventas por region
- Tendencia de ventas por mes

## Como ejecutarlo
1. Clonar el repositorio
2. Instalar dependencias: pip install pandas matplotlib seaborn
3. Ejecutar: python analisis.py
